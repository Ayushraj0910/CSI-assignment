## Week 1 — ML Foundations

Covered Python, NumPy, Pandas, Linear Algebra, Statistics, and Probability Theory.

### What I Learned
- **Python**: control flow, dicts/sets/list comprehensions, exception handling, lambdas
- **NumPy**: array reshaping, indexing/slicing, matrix multiplication vs. element-wise ops
- **Pandas**: Series vs. DataFrame, `loc`/`iloc`, groupby aggregation, handling missing data
- **Linear Algebra**: vector norms, why matrix multiplication isn't commutative, eigenvalues/eigenvectors, SVD and its link to PCA
- **Statistics**: descriptive vs. inferential stats, hypothesis testing (t-tests, p-values), error metrics (MAE/MSE/RMSE/R²), drift detection (PSI, concept vs. covariate drift)
- **Probability**: joint/conditional probability, common distributions (Normal, Binomial, Poisson), Bayes' Theorem, Central Limit Theorem

**Takeaway:** connected the math foundations to the tools (NumPy/Pandas) used in nearly every ML workflow.
